# VAC v1.2.3
Valorant Assistant Companion

## Features
- map randomizer
- team charactets randomizer
- challenges randomizer
- map whitelist
- characters type whitelist
- challenges count
- characters count

## Download
- go in release tab
- click on the .zip archive
- extract the archive
- launch VAC.exe
